## Concept

Dans le cadre du Hackathon de 48 heures, sur le thème de l'écologie. Travail réalisé par groupe de 5 personnes dans le cadre de notre formation de développement web et web mobile à la Wild Code School. 

Nous avons réalisé une application grâce à React et à Tailwindcss pour le design. L'utilisateur peut créer son profil personnalisé pour gérer ses actions écoresponsables, voir sa progression au fil des semaines, rechercher d'autres actions à accomplir au quotidien pour la planète et se comparer aux autres réutilisateurs.
